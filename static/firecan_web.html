About FIRECAN

Firecan is a collection of historical forest fires data in Canada
The data used by Firecan can be publicly accesed (links below)
I strongly suggest looking at the raw data for more information
Download and run FIRECAN locally: GITHUB

Filter Data
To display polygons, first click the map on the side bar
Input your desired filters and click "Filter Map" to display polygons
Due to server limitations, requests are limited to 8mb
Either focus your filters or reduce polygon tolerance to reduce file size

Polygon Tolerance
Polygons are drawn uses lists of coordiantes
To reduce load we can decrease the amounts of coordiantes used for each polgyon
In the filtering side bar click the button
Enter a lower number to draw a more detailed polygon
This number refers to allowed deviation when reducing the polygon

Watershed Explorer
Currently, firecan only supports watershed filtering in Quebec
To filter by watershed, first click on the watershed explorer
Choose your desired watershed, copy the name and past it into the watershed input box
Filting based on location in other polgyons takes a big longer than normal filtering

Download Data
Donwload your filtered data by entering your desired filters are pressing the save button
Select your file format and press save

Base Map
Change the basemap using the pop-up in the top right of the map.

Data Sources:

  • Quebec Fires: https://www.donneesquebec.ca/recherche/dataset/feux-de-foret
  • Quebec Watersheds: https://www.donneesquebec.ca/recherche/dataset/bassins-hydrographiques-multi-echelles-du-quebec
  • All other Provinces: https://cwfis.cfs.nrcan.gc.ca/datamart
